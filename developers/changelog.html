Changelog

A complete history of every MoltChain release — new features, improvements, breaking changes, and fixes.

v0.11.0 February 10, 2026

Production Hardening & Full WASM Build

New Features

  • AES-256-GCM authenticated encryption for CLI keypairs — replaces legacy XOR cipher with backwards-compatible version field
  • ZK proof verification via HMAC-SHA256 — 81-byte minimum proof structure with constant-time comparison
  • All 16/16 WASM smart contracts now compile and build successfully
  • Criterion benchmarks — 10 benchmarks across 3 groups (tx processing, block creation, signature verification); ~35K tx/s measured
  • CI: integration test job (validator + comprehensive test scripts) and WASM contract build job with caching
  • CONTRIBUTING.md — full developer guide with setup, testing, code quality, WASM builds, and PR process
  • Centralized frontend URL config (shared-config.js) — auto-detects dev vs production, replaces 15 hardcoded localhost URLs

Improvements

  • Faucet CORS: explicit origin allowlist replaces permissive policy — restricted to localhost dev ports + production domains
  • Wallet: fixed 4 async function declarations that caused SyntaxError breaking all wallet JS
  • Wallet CSS: added display:none/block for create-step and import-method (wizard step visibility)
  • Generated favicons and PWA icons for all 5 frontend apps

Fixes

  • Dockerfile binary name corrected to match Cargo.toml output
  • .dockerignore added to prevent 600MB+ context bloat
  • README.md: removed false GraphQL/gRPC claims
  • MIT LICENSE file added
  • cargo fmt applied project-wide
v0.10.0 February 10, 2026

MoltyID Universal Identity

New Features

  • .molt naming system — register, resolve, transfer, renew, and release human-readable names
  • Agent Discovery Registry with endpoints, metadata, availability signalling, and rate-limit config
  • Trust tiers (0–5) integrated into mempool priority lanes
  • Identity-gated access on MoltBridge, Compute Market, BountyBoard, ClawPay, and MoltSwap
  • Express lane in mempool for Tier 4+ agents — lower latency, higher throughput
  • 113 tests added across 7 previously-untested contracts

Improvements

  • 8 new fuzz targets covering consensus, block, mempool, account, hash, and RPC subsystems
  • Whitepaper updated with full MoltyID specification

Fixes

  • 134 clippy warnings resolved — project now compiles with zero warnings
v0.9.0 February 9, 2026

Production Readiness

New Features

  • 98/98 fix plan items completed — full production hardening pass
  • 5 vision contracts: reef_storage, moltbridge, compute_market, clawpay, bountyboard
  • SDK test_mock framework for contract testing without a running node
  • Parallel transaction processing scaffold for future throughput scaling
  • Privacy layer scaffold — encrypted memo fields and stealth address generation
  • Event streaming via WebSocket subscriptions

Fixes

  • WriteBatch atomicity enforced for all state mutations — no partial writes
  • Bootstrap sybil protection prevents address flooding during peer discovery
  • ReefStake RPC authentication — endpoints now require valid signatures
  • P2P TLS enforcement — plaintext connections rejected
  • EVM balance commit correctness — balances now round-trip through EVM without drift
  • ForkChoice with proper chain scoring — heaviest-chain rule replaces naive longest-chain
v0.8.0 February 5, 2026

Ecosystem Launch

New Features

  • Explorer (Reef Explorer) — full blockchain explorer with block, transaction, and account views
  • Marketplace (Molt Market) — NFT marketplace with listing, bidding, and royalties
  • Wallet (Molt Wallet) — browser wallet with BIP39 seed phrase and hardware key support
  • Website landing page with live on-chain stats
  • Programs Playground — web-based contract IDE with compile, deploy, and interact
  • JS/TS, Python, and Rust SDKs for full-stack application development
  • WASM compiler service — compile contracts server-side without local toolchain
v0.7.0 January 2026

Core Protocol

New Features

  • Initial blockchain with Proof of Contribution consensus mechanism
  • 11 smart contracts: moltcoin, moltdao, moltswap, moltyid, moltoracle, moltauction, moltmarket, moltpunks, clawpump, clawvault, lobsterlend
  • CLI with wallet, validator, and query commands
  • P2P networking with gossip protocol for block and transaction propagation
  • RPC server with JSON-RPC 2.0 compliant API
  • Docker deployment support for single-command node setup
