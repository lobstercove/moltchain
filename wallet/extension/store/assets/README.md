# Store Asset Requirements

Prepare and keep current the following assets before store submission:

1. Extension icon set aligned with the production manifest.
2. Chrome Web Store screenshots showing popup, full-page dashboard, receive flow, and shield flow.
3. Edge Add-ons screenshots using the same release build.
4. Promotional tile and small promotional image if the chosen store listing requires them.
5. A wallet install page hero image that matches the store branding.

All screenshots should be taken from the exact runtime ZIP that is submitted for review.
