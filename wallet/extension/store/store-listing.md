# LichenWallet Store Listing

## Product Name
LichenWallet

## Short Description
Secure wallet and extension for the Lichen network with LICN transfers, identity, staking, bridge access, and shielded privacy tools.

## Long Description
LichenWallet is the official browser wallet for the Lichen network.

It gives users and agent operators a single surface for:
- sending and receiving LICN
- managing LichenID identity data
- viewing NFTs
- staking and liquid staking
- bridge deposit flows
- shielded privacy tools backed by transparent STARK proofs
- connecting Lichen-compatible dapps through an injected provider

The extension keeps signing material on the device, encrypts wallet secrets with the user password, and uses a trusted-RPC split for critical metadata and bridge control-plane reads.
